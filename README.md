# Lean_DataMining
datamining Algorithms

## kmeans.py:
* 用k均值的方法进行聚类
  - 主要函数：
    + 根据分类计算聚类中心
    + 计算两点之间距离
    + 将所有点按照距离中心点的距离进行分类
    + 初始化：创建k个中心点
  - 算法整体思想：见https://blog.csdn.net/qq_36219266/article/details/82657694
## Apriori.py:
  - 主要函数：
    + 创建初始候选集
    + 计算支持度
    + 根据支持度筛选小于支持度的
    + 产生候选集
    + 计算置信度
  - 算法整体思想：见https://blog.csdn.net/qq_36219266/article/details/82707708
